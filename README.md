# Recipe Management System

A Java-based desktop application developed as a course project to manage recipes efficiently.  
This application allows users to store, organize, and manage recipe details through a simple interface.

## Features

- Add new recipes
- Update recipe details
- Delete recipes
- Search recipes
- View stored recipes
- Simple and user-friendly interface

## Technologies Used

- Java
- Eclipse IDE
- Object-Oriented Programming (OOP)

## Project Structure

```text
src/        -> Source code
bin/        -> Compiled files
```

## How to Run

1. Open the project in Eclipse IDE
2. Build the project
3. Run the main Java file

## Course Project

This project was developed as part of an academic course project.
